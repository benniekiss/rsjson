# Installation

With lux:

```shell
lx install rsjson
```

With luarocks:

```shell
luarocks install rsjson
```

With cargo, where `{version}` is one of `55`, `54`, `53`, `52`, or `51`.

```shell
cargo install rsjson --no-default-features --features module,lua{version}
```
